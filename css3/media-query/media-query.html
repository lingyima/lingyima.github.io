媒体特征

媒体特征 取值 接受min max 简介
width length yes 设备页面可见区域宽度
height length yes 设备页面可见区域高度
device-width length yes 设备屏幕可见宽度
device-height length yes 设备屏幕可见高度
orientation portrait|landscape no 定义height是否大于等于width.portrait是，landscape否
aspect-ratio ratio yes 定义width与height的比率
device-aspect-ratio ratio yes 定义device-width与device-height的比率
color integer yes 定义每一组输出设备的彩色原件个数。如果不是彩色设备，则值等于0
color-index integer yes 定义在输出设备的彩色查询表中的条目数。如果没有使用彩色查询表，则值等于0
monochrome integer yes 定义在一个单色框架缓冲区中每像素包含的单色原件个数。如果不是单色设备，则值等于0
resolution resolution yes 定义设备的分辨率：如：96dpi,300dpi,118dpcm
scan progressive|interlace no 定义电视类设备的扫描工序
grid integer no 用来查询输出设备是否使用栅格或点阵。只有1和0才是有效值，1代表是，0代表否
媒体类型 解释
all 所有设备
screen 彩色电脑屏幕
print 文档打印或打印预览模式
projection 项目演示，比如幻灯
projection 手持设备
speech 演讲
tv 电视
tty 固定字母间距的网络的媒体，自如电传打字机
braille 盲文
embossed 盲文打印
媒体查询规则/类型 解释
@media all and (min-width:800px) 所有最小水平屏幕宽度为 800 像素的屏幕应用规则
@media (min-width:800px) 简介方式，同上
@media (min-width:800px) and (max-width:1200px) 所有介于 800px 与 1200px 之间的屏幕应用规则
@media (min-width:800px) or (orientation:portrait) 宽度至少是 800px 或方向是纵向的，则会应用该规则
@media (not min-width:800px) 最小宽度不是 800px 时，则会应用该规则
@media (width:800px) and (height:400px) 屏幕正好是 800像素宽、400像素高时应用规则
@media (min-width:700px),handhelp and (orientation：landscape) 最小宽度为700像素，或是横屏幕的手持设备上，同时应用样式

屏幕尺寸

  • 老智能机器：320px-480px
  • 智能手机：>=480px
  • 平板电脑：>=768px
  • 中等屏幕（桌面显示器）：>=992px
  • 大屏幕（大桌面显示器）：>=1200px
